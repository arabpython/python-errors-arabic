---

## ماذا ستجد هنا؟

ستجد في هذا المستودع أمثلة مبسطة لأخطاء شائعة في بايثون، مثل:

- خطأ عدم العثور على مكتبة أو وحدة
- خطأ الوصول إلى عنصر غير موجود داخل القائمة
- خطأ تحويل النص إلى رقم
- خطأ دمج النصوص مع الأرقام
- خطأ استخدام دالة نصية على قائمة
- أخطاء القوائم
- أخطاء أنواع البيانات
- أخطاء الإدخال والتحويل
- أمثلة توضح الخطأ والحل الصحيح
